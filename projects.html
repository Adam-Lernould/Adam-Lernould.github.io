Description détaillée

Développement d'un scanner de réseau en Python utilisant des bibliothèques comme Scapy et Nmap. L'outil permet de découvrir les hôtes actifs sur un réseau, de scanner les ports ouverts, et d'identifier les services en cours d'exécution, fournissant une base pour l'audit de sécurité.

Compétences acquises

  • Scripting Python pour la cybersécurité
  • Analyse de paquets (Scapy)
  • Techniques de scan réseau
  • Automatisation de tâches de sécurité

Description détaillée

Configuration avancée de la sécurité d'un réseau multi-sites. Le projet a consisté à établir un tunnel VPN IPSEC sur GRE pour chiffrer les communications entre les sites, et à déployer DNSSEC pour protéger le service DNS contre les attaques de type "man-in-the-middle". L'ensemble a été validé en suivant les recommandations de l'ANSSI.

Compétences acquises

  • VPN (IPSEC, GRE)
  • Sécurité DNS (DNSSEC)
  • Durcissement de services (Web, DNS)
  • Conformité (ANSSI)
